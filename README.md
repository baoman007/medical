# 智慧医疗AI助手

## 项目简介

智慧医疗AI助手是一个符合中国医疗合规要求的企业级医疗服务AI Agent应用，提供智能问诊、报告解读、预约挂号、医生复核等功能。

## 核心功能

### 1. 智能问诊
- 症状分析与分诊建议
- 常见症状快速选择
- 健康咨询与科普
- 用药提醒与注意事项
- 紧急情况识别与警示

### 2. 报告解读
- 支持上传检查报告图片
- OCR识别报告内容
- AI辅助解读各项指标
- 生成通俗易懂的解释
- 提供健康建议

### 3. 预约挂号
- 科室选择（支持子科室）
- 医生选择与排班查询
- 就诊日期与时间选择
- 患者信息录入
- 预约成功确认

### 4. 医生复核
- 待复核任务列表
- AI建议详情查看
- 医生审核确认/驳回
- 审核意见记录
- 复核历史查询

### 5. 个人中心
- 个人信息管理
- 健康档案查看
- 预约记录管理
- 健康检查记录
- 隐私设置

## 合规说明

### 可实现功能（合规安全）
- ✅ 智能导诊、分诊
- ✅ 健康咨询、症状科普
- ✅ 用药提醒、复诊提醒
- ✅ 病历/检查报告解读（辅助，不诊断）
- ✅ 挂号、预约、缴费、查询流程助手
- ✅ 慢病管理、随访、健康宣教
- ✅ 医院内部：医嘱核对、排班、耗材管理

### 红线（绝对不能做）
- ❌ 不能下诊断
- ❌ 不能开处方
- ❌ 不能替代医生
- ❌ 不能做治疗方案推荐
- ❌ 不能处理急症

### 合规要求
- 所有对外产品必须标注：AI辅助，不构成医疗建议
- 数据不存储敏感病历
- 实施隐私计算与脱敏
- 符合《个人信息保护法》《医疗卫生机构网络安全管理办法》

## 架构说明

### 标准流程
```
用户输入 → 意图识别 → 工具调用 → RAG知识库 → 大模型生成 → 安全审核 → 输出
```

### 核心模块
1. **意图识别模块**：判断用户意图（症状咨询、报告解读、用药咨询等）
2. **工具调用能力**：调用HIS系统、PACS/LIS系统、用药知识库等
3. **RAG医疗知识库**：临床指南、药品说明书、疾病科普等
4. **医疗专属工作流**：症状咨询流程、报告解读流程等
5. **安全与合规**：四层审核机制

## 企业部署建议

1. 选择商用医疗大模型（通义医疗/讯飞星火/智谱GLM）
2. 搭建RAG知识库（医院资料+权威指南）
3. 实现3个核心工具：挂号、报告解读、用药咨询
4. 建立安全审核模块
5. 开发前端小程序/Web
6. 内部测试 → 医生审核 → 正式上线
